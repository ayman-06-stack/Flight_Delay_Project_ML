# ✈️ Flight Delay Prediction - Machine Learning Project

Prédiction intelligente des retards de vols en utilisant le Machine Learning.

## 📋 Description

Ce projet utilise des algorithmes de Machine Learning pour prédire avec précision les retards de vols. Il analyse les données historiques et les patterns pour fournir des prédictions fiables.

## 🎯 Objectifs

- ✅ Prédire les retards de vols avec haute précision
- ✅ Analyser les facteurs impactant les retards
- ✅ Créer un modèle ML performant et scalable

## 🛠️ Technologie utilisée

- **Python 3.x**
- **Pandas**: Manipulation des données
- **Scikit-learn**: Modèles ML
- **NumPy**: Calculs scientifiques
- **Matplotlib/Seaborn**: Visualisation
